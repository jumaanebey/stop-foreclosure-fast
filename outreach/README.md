# Email Outreach Templates

Email sequences and referral partner templates for lead nurturing and professional networking.

## Files

### email-sequences.md

Four automated drip campaigns for different lead types:

| Sequence | Trigger | Emails | Duration |
|----------|---------|--------|----------|
| 1. New Lead | Form submission | 5 emails | 14 days |
| 2. Guide Download | PDF download | 3 emails | 5 days |
| 3. Post-Consultation | After call (no deal) | 3 emails | 7 days |
| 4. NOD Mailer Follow-Up | Direct mail recipient | 3 emails | 10 days |

Each sequence includes subject lines, body copy with merge fields (`{{first_name}}`, `{{property_address}}`), and optimal send timing.

### referral-partner-emails.md

Cold outreach templates for 7 referral partner types:

1. Divorce Attorneys
2. Estate/Probate Attorneys
3. Real Estate Agents
4. Bankruptcy Attorneys
5. Financial Advisors / CPAs
6. Churches / Community Organizations
7. HUD Counseling Agencies

Also includes a follow-up template and LinkedIn connection request.

## Using with Email Tools

**Mailchimp / ConvertKit / ActiveCampaign:**
1. Create automation for each sequence trigger
2. Copy email content and set delays between sends
3. Replace `{{merge_fields}}` with platform syntax
4. Add unsubscribe links as required

**Gmail / Manual:**
1. Save templates as Gmail canned responses
2. Set calendar reminders for follow-up sends
3. Personalize merge fields before sending
