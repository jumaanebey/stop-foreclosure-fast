## Script Format

Each script contains:

- **Video metadata** - length, target audience, tone
- **Timestamped sections** - HOOK, INTRO, main content, CTA
- **On-camera direction** - delivery notes, eye contact cues
- **B-roll suggestions** - visual content needed for each section
- **Graphics list** - title cards, callouts, lower thirds
- **Production notes** - tone reminders, filming tips
